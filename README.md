1. For running code on a machine from a venv, you must first allow your computer to use them. You can do this with: 
```Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser```
2. After obtaining all the files, run "python launch.py"
3. In VSCode, go to the bar up top and write: ">Python: Select Interpreter"
4. Select .venv/... as the interpreter
5. Run any file
